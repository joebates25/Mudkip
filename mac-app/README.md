# MarkdownViewerMac

Native macOS prototype using SwiftUI + WKWebView.

## Run

```bash
cd /Users/josephbates/dev/markdown-viewer-poc/mac-app
swift run MarkdownViewerMac
```

## Open a file on launch

```bash
cd /Users/josephbates/dev/markdown-viewer-poc/mac-app
swift run MarkdownViewerMac /Users/josephbates/dev/tinycc/study_guide/04_parsing_symbols.md
```

Inside the app, use `Open Markdown File` or `Cmd+O` for the file picker.
